# CodeAlpha Tasks - Language Translation Tool

## Task 1: AI Translation Tool
Un traducteur intelligent utilisant l'IA avec une interface style ChatGPT.

### Fonctionnalités
* 🎤 **Voice Input** : Reconnaissance vocale intégrée.
* 🔊 **Text-to-Speech** : Écoute de la traduction.
* 🕒 **History** : Sauvegarde locale des dernières recherches.
* ⚡ **Performance** : Affichage du temps de réponse et gestion du cache.

### Installation
1. Clonez le projet.
2. `npm install` pour les dépendances.
3. Créez un fichier `.env` avec votre clé API.
4. `node server.js` pour lancer le backend.
